Power BI Dashboard
For Shipping Company

A dashboard was created to fulfill the shipping company's needs. This useful tool shows all necessary information about routes, vessels, errors, orders, and more. The dashboard serves as an assistant to employees, allowing them to answer various questions much more quickly than before because the answers are just a few clicks away.

1. Company provided an Excel file with data. All neccessary data manipulations were done in Power Query Editor (replacing errors and values, removing columns, merging and grouping).

2. Creation of data model with relationships between tables. (adding calculated columns, creating calendar table, joining tables with relationships)

3. Creation of measures with DAX. There are over 30 measures that were created for this dashboard.

4. Data visualizations with various visuals (card, bar chart, line chart, map, table, etc).

5. Useable dashboard with links between pages, filters, slicers and buttons.
