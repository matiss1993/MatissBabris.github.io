Step by step guide

Short guide on how to achieve this kind of data visualization. You can use 2 videos I created as simple tutorials and read a brief step by step guide. All you need is:
1) Geo-package file (.gpkg) with data. I used data from Kontur;
2) QGIS software;
3) Blender 3.5;
4) Any video editing software or online tool that can create GIF from images. I used Adobe Premiere Pro.

Step 1: Download data and upload it in QGIS

  • Download .gpkg file;
  • Open QGIS software;
  • Click on Open Data Source Manager button or use Ctrl+L shortcut;
  • Choose Vector on left-side menu of Data Source Manager and in source browse for .gpkg file that you downloaded;
  • Click on Add, select Population item and press Add Layers button;
  • Now you should see your data in front of you and it is formed from hexagons.

Step 2: Create a displacement map

In order to create this displacement map, we need to create 2 additional calculated columns in data: area of each hexagon and population density in each hexagon.
  • Right click on created layer, chosse Open Attribute Table;
  • Press on Open Field Calculator button or use Ctrl+I shortcut;
  • Create Area column. In Output field name enter the name of column. In this case Area is a good option. In expression field enter this: `$area / 1000000` to convert the value to km2, click OK;
  • Create Population Density column. Do as in previous step, just change the column name to Density and expression to: `"population"/"Area"`;
  • Now to create the gradient map of density, right click on layer, choose Properties and click on Symbology;
  • In drop-down menu choose Graduated, change Value to Density.
  • In color ramp choose light grey as Color 1 and Black as Color 2. And it should be Continuous not Discrete.
  • In Mode choose Natural Breaks (Jenks) or Even Intervals option and in classes you can choose how many "bins" to create.
  • In Symbol, click on the color, Simple Fill and set stroke style to No line.
  • Now you should have something similar to a grayscale map.

Step 3: Export image

  • Project -> Import/Export -> Export Map to Image
  • Adjust image dimensions and resolution as necessary. I created image of 16710 x 16710 px and 1200 dpi. Choose file location and export it.

Step 4: Creating a 3D model

  • Open up a Blender software
  • Press A to select everything in scene, press X to delete selection and press Enter to confirm.
  • Shift+A -> Mesh -> Plane.
  • S -> 15 -> Enter to scale the plane 15 times bigger.
  • Press Tab to enter the Edit Mode. Right-click on plane and choose Subdivide. Shift+R to repeat. (I did it for 8-9 times)
  • Press Tab to leab Edit Mode and enter Object Mode. Add Displace modifier. Add new texture and choose the image that was created in QGIS.
  • Change Strength to -0.2 and Midlevel to 1.0
  • Add Subdivision Surface modifier, place it before Displace modifier and change Levels Viewport to 2
  • Open Shader Editor and create new material named Density. Add ColorRamp, Seperate XYZ and Texture Coordinates nodes.
  • Texture Coordinates Object to Seperate XYZ Fac, Seperate XYZ Z to ColorRamp Fac, ColorRamp Color to Principled BSDF Base Color.
  • Adjust ColorRamp settings as you desire, to achieve your goal.
  • Add Camera with path.
  • Add Area lights.
  • Customize light, camera and animation settings.
  • Render Animation.

Step 5: Creating a GIF

  • Find a video software or online tool that can convert images to GIF.
  • Upload rendered images and create GIF
