Online Retail Business
Analysis with Python

Utilizing Python's libraries, including Matplotlib, Pandas, and Numpy, to perform data loading and cleaning, conduct an Exploratory Data Analysis (EDA), and create informative visualizations, while analyzing a Kaggle dataset concerning online business retail sales. .

I used dataset about Online Retail Business that I found on Kaggle.

Analysis was conducted in Python, using VSCode code editor. I created Jupyter Notebook file in which you can see not only the code, but also the outputs.

Main Python libraries that were used in this project:
1. Pandas - for data cleaning, data manipulations (grouping, aggregating, adding new columns);
2. Matplotlib - for data visualizations;
3. Numpy, Scypy - for some numerical manipulations and statistics
